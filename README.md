# ELNET Events & Travel - Corporate Website

A complete, responsive corporate website for **ELNET Events & Travel**, a division of Elnet Technology Plc. The site showcases event management services, travel destinations across Ethiopia, a car rental fleet, and city tour packages.

## 🔗 Live Demo

- **Vercel:** https://elnet-events-travel.vercel.app
- **GitHub Pages:** https://tsi777.github.io/elnet-events-travel/

## 📋 Project Overview

This website was built to establish a premium online presence for ELNET Events & Travel, a professional event and travel management company operating in Ethiopia. The design reflects a **corporate, diplomatic, and luxurious** brand identity with a custom navy and gold color scheme.

### Key Pages

| Page | Description |
|------|-------------|
| **Home** | Hero section, company overview, services preview |
| **Events** | 8 service categories + 3-phase event framework |
| **Travel** | Southern, Northern, Eastern, and Specialized tours |
| **Addis Tours** | 5 city tour packages + bonus attractions |
| **Fleet** | Car rental fleet (VIP, Executive, Standard, Minibuses) |
| **Contact** | Office info + inquiry form |

## 🛠️ Technologies Used

| Category | Technologies |
|----------|--------------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Design** | Playfair Display (headings), Inter (body text) |
| **Icons** | Font Awesome |
| **Version Control** | Git, GitHub |
| **Hosting** | GitHub Pages, Vercel |

## ✨ Key Features

- Fully responsive for desktop, tablet, and mobile
- 6 interconnected pages with seamless navigation
- Custom navy (#0E1A3A) and gold (#B59D12) brand colors
- Interactive contact form with validation
- Optimized images for fast loading
- Smooth scroll navigation
